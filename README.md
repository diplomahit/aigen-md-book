# Markdown 完全指南 —— 从语法到生产级文档的轻量之道

一本系统讲解 Markdown 语法、生态与生产实践的开源电子书。

## 目录

本书共四篇，15 章，涵盖从入门到实战的完整知识体系：

### 篇一：Markdown 的直觉（基础篇）

| 章节 | 内容 |
|------|------|
| 第1章 | 为什么是 Markdown？ |
| 第2章 | 行内语法全解 |
| 第3章 | 块级元素全解 |

### 篇二：Markdown 的进阶（提升篇）

| 章节 | 内容 |

| 第6章 | 图片与嵌入 |
| 第7章 | 脚注与自定义属性 |

### 篇三：Markdown 的生态（工具篇）

| 章节 | 内容 |
|------|------|
| 第8章 | Markdown 引擎揭秘 |

### 篇四：Markdown 在生产环境（实战篇）

| 章节 | 内容 |
|------|------|

## 项目结构

```
aigen_md_book/
├── outline.md          # 全书大纲
├── details/            # 各章详细笔记
├── articles/           # 各章正文（含图片和资源）
│   ├── assets/images/  # 插图资源
│   ├── 1_body.md       # 第1章正文
│   ├── ...
```

## 如何阅读

- **大纲**：先阅读 `outline.md` 了解全书结构
- **按章阅读**：进入 `articles/` 目录，按编号阅读各章正文
- **详细笔记**：进入 `details/` 目录获取要点总结
